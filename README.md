# Guitar Ranked Supabase

## Variables Vercel à ajouter

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Supabase

1. Crée un projet Supabase.
2. Va dans SQL Editor et exécute `supabase-schema.sql`.
3. Va dans Authentication > Users > Add user et crée ton compte admin.
4. Mets les variables dans Vercel puis redeploy.
